# India_Census-2011

India Census 2011 Data Analysis & Dashboard
Project Overview
This project analyzes the India Census 2011 dataset to understand population, literacy rate, sex ratio, religion distribution, and work participation across different states and districts in India.
The project includes:
•	Data analysis using Python
•	Data visualization using Matplotlib
•	Interactive dashboard using Power BI

Tools Used
•	Python
•	Pandas
•	Matplotlib
•	Power BI

Python Analysis
In Python, the following tasks were performed:
•	Data loading and exploration
•	Data cleaning and checking missing values
•	Feature engineering (Literacy Rate, Sex Ratio)
•	Filtering districts based on population and literacy
•	Aggregation by state
•	Data visualization
Charts created:
•	Top 10 States by Population
•	Top Districts by Population
•	Literacy Rate Distribution
•	Male vs Female Population

Power BI Dashboard
The Power BI dashboard shows:
•	Total Population
•	Total Districts
•	Literacy Rate %
•	Sex Ratio
•	Work Participation Rate
•	Top 10 States by Population
•	Top Literacy States
•	Religion Distribution by State
•	Age Group Distribution
Filters for States and Districts are also included for interactive analysis.
